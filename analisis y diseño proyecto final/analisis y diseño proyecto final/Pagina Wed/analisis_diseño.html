Informacion de la app de registro de asistencia

Descripcion

Esta app es un sotware libre para colegios, el sotware contara con un registro de estudiantes matriculados,asignacion de grados, asistencia de estudiantes,un login para el auxiliar director o profesor , tambien contara con un registro de login donde se puede crear su contraeña y usuario


¿Que buscamos resolver?

Tomar asistencia manualmente no es tan optimo y se pueden perder los registros, con nuestro sowtare buscamos soluciona ese problema



funcionamiento del aplicativo


login

El Login ahi tenemos 2 botones Iniciar secion y Registrar, el boton registrar es para crear nuestro usuario y contraseña y el boton iniciar sesion para entrar al menu principal del aplicativo



registro

Se llenaran los datos para crear el usuario y contraseña
1. El boton Nuevo sirve para limpiar todas los campos
2. El boton guardar sirve para guardar nuestros datos y tener nuestra contrasea y usuario q ingresamos
3. El boton Modificar sirve para editar
4. El boton Eliminar sirve para eliminar un usuario

La tabla servira para administrar alos usuarios y contraseñas



Menu principal

En el menu se podra seleccionar:
1.Registro Alumno
2.Registro grados
3.Registro de Asistencia
4.Registro de matriculados
5.Cerrar Sesion



Creacion de grado

Creamos grados y secion para incorporar alos alumnos ahi



Registro estudiante

Se llenan los datos del formulario y se guarda para registrar al alumno , tambien se puede modificar y eliminar al alumno



Registro Matricula

una vez registrado el estudiante, se le da su Grado y sesion



Registro de Asistencia

se selecciona al estudiante y se le toma la asistencia
